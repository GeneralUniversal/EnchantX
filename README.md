## 增强

本项目使用MCAPI

- 允许附魔超过或低于原版等级 (范围:-32767~32767)



## 安装方法

```cmd
lip install github.com/GeneralUniversal/EnchantX
```

## 使用

指令操作目标选择器手持物品

- /enchant <玩家> <附魔ID:数字> <附魔等级:数字> 添加附魔
- /enchant <玩家> <附魔英文:字符串> <附魔等级:数字> 添加附魔
